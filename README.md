# 📈 MOEX Portfolio — Инвестиционный проект Дмитрия

Персональная система управления инвестиционным портфелем на Московской бирже.

## 🎯 Цель проекта
Отслеживать портфель акций, обучаться инвестициям и получать аналитику — всё в одном месте.

## 👤 Роли в проекте

### 🎓 Ментор
Обучающая роль. Выдаёт статьи и уроки о том, как работает фондовый рынок, что такое дивиденды, как читать графики и принимать решения. Периодичность: раз в 1–2 дня.

### 📊 Аналитик
Инвестиционная роль. Отслеживает новости, макроэкономику и факторы, влияющие на акции в портфеле. Даёт честную сводку — без домыслов, только реальные данные.
